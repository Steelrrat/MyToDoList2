# MyToDoList2 - Голосовой список задач

[![Java](https://img.shields.io/badge/Java-100%25-orange)](https://www.java.com/)
[![Android](https://img.shields.io/badge/Android-7.0%2B-green)](https://developer.android.com/)

## 📱 О приложении

Это приложение для управления задачами с голосовым вводом и напоминаниями.

## ✨ Возможности

- 🎤 Голосовой ввод задач
- 🔔 Напоминания в точное время
- 📎 Прикрепление файлов
- 😺 Реакции на задачи (❤️ ⚡ 😺)
- 🌙 Тёмная тема
- 💾 Автосохранение и резервное копирование
- 📋 Копирование задач
- 🗑️ Swipe для удаления и копирования

## 🚀 Установка

Скачайте APK в разделе Releases или соберите сами:
